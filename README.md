# Members Only

# Overview
This express project tests my authentication and verification skills using express-validator, passport, express-session and bcyptjs. It has two models called users and messages where users write messages to a private club that they can only access by providing a secret code. 

# Features
- All users can see and post messages listed on the homepage
- Users can sign up and choose if they are admin or not
- Only club members can see message author names and post creation dates
- Only club admin members can see the delete-message button and delete messages
- Users can join the private club by providing a secret passcode
- Users can log in and log out
- The navbar changes based on logged-in status
